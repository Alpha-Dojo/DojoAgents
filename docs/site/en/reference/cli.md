# CLI Reference

The CLI parser is defined in `dojoagents/cli/main.py`.

| Command | Purpose |
| --- | --- |
| `chat` | Run a local agent request |
| `dashboard` | Start the dashboard |
| `gateway` | Start the gateway |
| `gateway setup` | Configure an adapter |
| `gateway pairing` | Manage pairing |
| `scheduler` | Load scheduled jobs |
| `model` | Configure model provider |
| `mcp serve` | Start MCP server |
| `precompute-sector` | Precompute sector data |
