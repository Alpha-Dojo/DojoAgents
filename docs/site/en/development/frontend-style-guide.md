# Frontend Style Guide

Dashboard UI should be quiet, clear, and scan-friendly. Prefer familiar controls, stable layout dimensions, readable typography, and restrained palettes.

Core rules:

- Use cards only for repeated items, modals, or framed tools.
- Keep page sections unframed.
- Avoid text overflow on mobile and desktop.
- Use existing frontend conventions before inventing new ones.

Detailed frontend rules should be expanded in this page over time.
