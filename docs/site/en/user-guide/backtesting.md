# Backtesting

Backtesting is intended to fetch historical data from DojoSDK or local stores, run local or remote backtest jobs, persist results, and visualize equity curves.

## Current Direction

- Data comes from DojoSDK or local stores.
- Execution can be local or remote.
- Local results should be persisted for frontend queries.
- Remote results are fetched after task completion.
- Visualization focuses on equity curves, drawdown, returns, and key metrics.
