# Skills

Skills provide procedural knowledge, tool instructions, and workflow guidance for agents.

## Modules

| Module | Purpose |
| --- | --- |
| `dojoagents/skills/loader.py` | Load skill files |
| `dojoagents/skills/cache.py` | Cache skill content |
| `dojoagents/skills/manager.py` | Manage skill lifecycle |
| `dojoagents/tools/skill_manage.py` | Expose skill management tools |

Built-in skills live under `dojoagents/skills/built_in/`. External skills are usually discovered through plugins or user directories.
